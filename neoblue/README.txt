#######################
INSTALLATION
#######################
1)Copy the "neoblue" directory to
your_site/themes/

2)Upload the theme's neoblue directory to the webserver's XOOPS_ROOT/themes/ directory

3)Enable the "neoblue" theme trough your site General settings
for both "Default Theme" and "Admin Theme"

4)Enjoy!

If you have troubles/requests contact me at
xoops site www.xoops.org by private message
or,better,post in xoops.org's support forums
http://www.xoops.org/modules/newbb/
